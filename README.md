# Sprachressourcen Analysetool

## Übersicht
Dieses Projekt ist ein einfaches Open-Source-Tool zur Analyse und Visualisierung von Sprachressourcen. Es ermöglicht Benutzern, verschiedene Datensätze zu laden, Sprachmerkmale zu extrahieren und sie in interaktiven Grafiken darzustellen.

## Funktionen
- Hochladen von Textdaten in gängigen Formaten (z.B. CSV, TXT)
- Analyse der sprachlichen Vielfalt und Struktur
- Visualisierung der Ergebnisse mit Diagrammen und Grafiken
- Export der Ergebnisse in gängige Formate

## Installation
1. Klone das Repository:
   ```bash
   git clone https://github.com/deinbenutzername/sprachresourcen-analysetool.git
   ```
2. Installiere die benötigten Abhängigkeiten:
   ```bash
   pip install -r requirements.txt
   ```

## Verwendung
Führe das Hauptskript aus:
```bash
python app.py
```
